# 天猫校园去广告

基于 **LSPosed (libxposed API 101)** 的天猫校园广告移除与精简模块。

## 功能一览

| 功能 | 说明 | 默认 |
|------|------|------|
| 去开屏广告 | 移除 App 启动时的全屏广告 | ✅ 开启 |
| 跳过启动页 | 冷启动/热启动均直接进入首页 | ✅ 开启 |
| 精简首页 | 仅保留**扫一扫、校园洗烘、校园吹风** | ✅ 开启 |
| 隐藏底部导航 | 移除商城、找搭子、小红花等多余 Tab | ✅ 开启 |
| 隐藏 AI 入口 | 去掉首页顶部「长按说话」按钮 | ✅ 开启 |
| 隐藏权益广告 | 移除会员权益卡片和广告容器 | ✅ 开启 |
| 屏蔽更新 | 阻止热更新及强制更新弹窗 | ✅ 开启 |

所有功能均可在模块内设置界面中单独开关。

## 运行要求

- **Root** 权限（Magisk / KernelSU / APatch 均可）
- **LSPosed 框架**（libxposed API 101+）
- 天猫校园 App `com.tmall.campus.and`（已测试版本：5.0.0）

## 安装与使用

1. 确保设备已 Root 并安装 LSPosed 框架
2. 下载 [天猫校园去广告_1.0_flow.apk](天猫校园去广告_1.0_flow.apk) 并安装
3. 打开 LSPosed → 模块 → 启用「天猫校园去广告」
4. 勾选作用域：**天猫校园**（`com.tmall.campus.and`）
5. 强制停止天猫校园后重新打开即可生效
6. 打开本模块 App 可查看激活状态，进入设置界面可自定义开关
